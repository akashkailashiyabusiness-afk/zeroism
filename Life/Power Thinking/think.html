Mentally Powerful

Apply a latticework of Munger-style mental models — invert, incentives, probabilities, circle of competence — and run thought experiments using your local LLM.

Active heuristics: Invert problems, check incentives, ask for first principles, estimate probabilities, limit within circle-of-competence, simplify.
Quick notes: Configure your local LLM Studio / server to accept POST requests. This app will POST JSON { model, prompt, temperature } or { prompt } depending on your server. See the "How to run" below.
How to run
  1. Start your local LLM (LM Studio / Ollama / local API).
  2. Point Local API URL to your generate/chat endpoint (example: http://127.0.0.1:8080/v1/generate or http://localhost:8080/v1/chat/completions).
  3. Press Test connection to verify. If CORS blocks in browser, run the file from a local server (python -m http.server) or use a CORS proxy for testing.
